# Modern Portfolio / Digital Agency Website

A sleek, responsive portfolio and digital agency website built with HTML5, Bootstrap 5, CSS3, and vanilla JavaScript. This template features a dark theme with neon accents, smooth animations, and a modern design.

## Features

- **Fully Responsive**: Works on all devices and screen sizes
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Dark Theme**: Eye-friendly dark mode with neon accents
- **Optimized Performance**: Fast loading and optimized assets
- **SEO Friendly**: Proper HTML5 semantic markup
- **Contact Form**: Ready-to-use contact form
- **Portfolio Showcase**: Beautiful project showcase section
- **Testimonials**: Client testimonials slider
- **Skills Section**: Animated skills progress bars
- **Blog Section**: Ready for your latest articles

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/portfolio-website.git
   cd portfolio-website
   ```

2. **Open in your browser**
   Simply open the `index.html` file in your preferred web browser.

## Customization

### Colors
Edit the CSS variables in `assets/css/style.css` to change the color scheme:

```css
:root {
  --primary: #8B5CF6;
  --secondary: #22D3EE;
  --dark: #0B0616;
  /* ... other variables ... */
}
```

### Content
- Update the content in `index.html`
- Replace placeholder images in the `assets/images/` directory
- Update social media links in the footer
